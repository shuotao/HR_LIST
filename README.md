# HRMD — 104 履歷自動化解析與人才篩選系統

專為人資主管與徵才團隊設計：從 104 人力銀行匯出的 PDF 履歷，經過自動化解析、資料清洗、規則篩選，產出符合職缺需求的候選人名單。

---

## 系統架構

本專案包含兩大 Agent 技能，各自獨立運作但可串接使用：

```
PDF 履歷 ──► [hr-resume-parser] ──► HR_Data_Summary.csv（結構化資料）
                                          │
ANALYSIS.md ──► [hr-talent-screener] ──► 候選人名單 + 命中理由
（104 系統擷取）     │                        │
                     ▼                        ▼
               三階段清洗              疊代學習（使用者回饋 → 規則精煉）
```

---

## 技能一：hr-resume-parser（履歷解析）

將個別候選人的 PDF 履歷轉為結構化 CSV。

**核心功能：**
- 全自動 PDF → Markdown → CSV 轉換，解決康熙部首亂碼與編碼問題
- 精準擷取 8 大欄位：姓名、年紀、語文能力、學歷、近期工作、工作內容、總年資、前二次任職公司
- 語文能力智能簡化（英文置頂、檢定成績附帶）
- 雙重防幻覺驗證（程式自動抽檢 5 人 + Agent 手動抽驗 5 人）

**執行方式：**
```bash
python scripts/convert_pdfs.py        # PDF → Markdown
python scripts/extract_hr_data.py     # Markdown → CSV
```

## 技能二：hr-talent-screener（人才篩選）

從 104 系統擷取的大量候選人資料（ANALYSIS.md）中，自動篩選出符合機電/廠務/工程職缺的面試候選人。

**核心功能：**
- **三階段資料清洗**：雜訊移除 → 代碼去重 → 學歷分類排序
- **規則化評分引擎**：必要條件（M1-M3）+ 加分條件（N1-N16）+ 排除條件（E1-E3）
- **疊代學習機制**：每批次根據使用者回饋（漏選/誤選）精煉篩選規則

**執行方式：**
```bash
python scripts/pipeline_clean.py ANALYSIS.md       # 三階段清洗
python scripts/screen_candidates.py ANALYSIS.md     # 評分篩選
```

**篩選規則概要：**

| 類型 | 說明 |
|------|------|
| 必要條件 (M) | 職稱含機電/廠務/監造等、有 EPC/營造/半導體經歷、3年以上年資 |
| 加分條件 (N) | 學歷對口、知名企業、管理職、多系統覆蓋、品管、能源工程等 |
| 排除條件 (E) | 純保全/門市/餐飲且無轉型跡象 |

## 注意事項

- **個資保護**：本工具建議於企業內網環境使用。所有範例人名須模糊化處理。
- **編碼規範**：CSV 採 `utf-8-sig` 編碼，可直接以 Excel 開啟。
- **Python 環境**：使用專案指定的綠色版 Python，不依賴系統全域安裝。
- **疊代精煉**：每次執行篩選後，應根據使用者回饋更新 `screening_rules.md`，讓下一次篩選更精準。
