# HR Resume AI Parser (104 履歷自動化解析工具)

🚀 **專為人資主管與徵才團隊設計**：將雜亂的 PDF 履歷轉化為精準的結構化數據，大幅節省人工查閱時間。

## 🌟 工具核心價值
1. **全自動轉換**：將 104 匯出的 PDF 履歷自動轉為 Markdown 文字，解決編碼混亂問題。
2. **精準欄位抽取**：自動擷取「年資、年紀、學歷、語文能力、近期工作」等關鍵數據。
3. **語文能力智能優化**：
   - 自動將「聽/精通、說/精通...」等冗長描述簡化。
   - **英文置頂顯示**：將最重要的語言能力放在首位，並附帶檢定成績（TOEIC/GEPT）。
4. **雙重防幻覺驗證**：
   - **程式自動校對**：腳本會回頭比對原始文字，確保姓名與數字無誤。
   - **Agent 人工抽驗**：結合 AI 技術對 PDF 原始檔進行掃描確認。

## 🛠️ 如何執行
1. **PDF 轉檔**：
   ```bash
   python scripts/convert_pdfs.py
   ```
2. **資料擷取與 CSV 產出**：
   ```bash
   python scripts/extract_hr_data.py
   ```

## 📝 注意事項
- **個資保護**：本工具建議於企業內網環境使用。開發測試階段請務必將範例人名模糊化（如上述範例）。
- **編碼規範**：產出的 CSV 採 `utf-8-sig` 編碼，可直接使用 Excel 開啟，不會出現亂碼。
