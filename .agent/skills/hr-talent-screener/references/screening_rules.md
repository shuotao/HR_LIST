# 人才篩選規則手冊 (Screening Rules)

> 本文件為跨批次有效的**純規則定義**。
> 不包含任何特定批次的人名、案例或統計數據。
> 每次疊代回饋後，僅更新規則本身。

---

## 一、目標職缺畫像

### 1.1 公司與部門定位
- **公司屬性**：EPC 統包工程公司（如中鼎集團）
- **部門需求方向**：機電工程（MEP）、廠務系統、建廠/擴廠專案
- **工作場域**：半導體廠房、石化/化工廠、商辦大樓、公共工程、資料中心等

### 1.2 執掌範圍
- 空調/HVAC 系統設計、施工與監造
- 消防系統設計與管理
- 電力系統（高低壓配電）規劃與維護
- 給排水/純水/廢水系統
- 廠務公用系統（CDA、PCW、PV、冷卻水塔、冰水主機等）
- 管線工程（LNG、製程管線、配管設計）
- 工程專案管理（發包、監工、驗收、預算控管）
- BIM 整合與施工圖繪製

## 二、篩選規則

### 2.1 必要條件（Must Have）—— 至少符合一項

| 編號 | 條件描述 | 判斷依據 |
|------|---------|---------|
| M1 | 近期職稱包含機電/廠務/空調/水處理/監造相關 | 搜尋「希望職稱」或工作經歷行 |
| M2 | 工作經歷中出現 EPC/營造/建廠/擴廠/半導體廠務 | 搜尋公司名或職稱 |
| M3 | 有 3 年以上機電/廠務/工程相關實務年資 | 觀察工作經歷行數與年份差 |

### 2.2 加分條件（Nice to Have）—— 符合越多越優先

| 編號 | 條件描述 | 權重 |
|------|---------|------|
| N1 | 學歷為電機/機械/冷凍空調/化工/環工/土木/建築 | ★★★ |
| N2 | 曾任職於知名 EPC 公司（中鼎、泰興、達欣、潤弘等）| ★★★ |
| N3 | 曾任職於半導體/高科技廠務（台積、世界先進、美光等）| ★★★ |
| N4 | 職稱含「主任」「課長」「副理」「經理」以上管理職 | ★★☆ |
| N5 | 工作內容涵蓋多系統（空調+消防+電力+給排水至少2項）| ★★☆ |
| N6 | 具備 BIM/Revit/AutoCAD 繪圖能力 | ★★☆ |
| N7 | 有監造/監工/品管經驗 | ★★☆ |
| N8 | 有新建廠/擴建廠經驗 | ★★☆ |
| N9 | 年資 6~20 年的中堅技術人員 | ★★☆ |
| N10 | 有管線工程（配管/LNG/製程管線）經驗 | ★☆☆ |
| N11 | 有 PLC/DCS/SCADA 自動控制經驗 | ★☆☆ |
| N12 | 具節能/ESG/ISO相關管理能力 | ★☆☆ |

### 2.3 排除條件（Exclusion）—— 符合任一項即排除

| 編號 | 條件描述 |
|------|---------|
| E1 | 經歷純粹為保全/門市/餐飲/銷售且無轉型跡象 |
| E2 | 希望職稱完全與工程/技術/機電/廠務無關 |
| E3 | 總工作年資不足 1 年且非應屆相關科系畢業 |

## 四、篩選經驗法則

> 以下為歷次疊代中累積的觀察與經驗，用於輔助 Agent 做軟性判斷。

### 4.1 學歷 vs 經驗
- **學歷非必要門檻**。非本科（如餐旅、資管、商業經營）但實務轉型成功者，只要有 5 年以上機電/廠務實戰經驗，仍可入選。
- 核心原則：**能力 > 學歷**。

### 4.2 年齡
- 不設嚴格年齡上限。歷史上 20~70 歲皆有被選入記錄。
- 但 **40~49 歲**為需求最大宗（約佔 36%），屬中堅技術人才。

### 4.3 年資甜蜜區
- **6~20 年**為最理想區間（約佔總選人之 68%）。
- 低年資（1-3 年）需具備「名校相關科系 + 半導體廠實習/正職」才有機會。
- 高年資（20 年+）需具備管理職或專家級經歷。

### 4.4 跨域整合價值
- 土木/建築背景但已轉型做機電、工地協調角色的人選具有獨特價值。
- 這類人適合「跨領域整合者」角色——能同時看懂土建圖與 MEP 圖。
