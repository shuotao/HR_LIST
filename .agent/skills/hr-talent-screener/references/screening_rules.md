# 人才篩選規則手冊 (Screening Rules)

> 本文件為跨批次有效的**純規則定義**。
> 不包含任何特定批次的人名、案例或統計數據。
> 每次疊代回饋後，僅更新規則本身。

---

## 一、目標職缺畫像

### 1.1 公司與部門定位
- **公司屬性**：EPC 統包工程公司（如中鼎集團）
- **部門需求方向**：機電工程（MEP）、廠務系統、建廠/擴廠專案
- **工作場域**：半導體廠房、石化/化工廠、商辦大樓、公共工程、資料中心等

### 1.2 執掌範圍
- 空調/HVAC 系統設計、施工與監造
- 消防系統設計與管理
- 電力系統（高低壓配電）規劃與維護
- 給排水/純水/廢水系統
- 廠務公用系統（CDA、PCW、PV、冷卻水塔、冰水主機等）
- 管線工程（LNG、製程管線、配管設計）
- 工程專案管理（發包、監工、驗收、預算控管）
- BIM 整合與施工圖繪製
- 機電採購發包（標單製作、邀標、議價、合約管理）
- 工程品管（品質查驗、材料品管、計畫書編寫）
- 能源工程（太陽能、儲能系統、充電樁、高低壓電力維運）
- 鋼構工程（焊接、鋼結構施工）

## 二、篩選規則

### 2.1 必要條件（Must Have）—— 至少符合一項

| 編號 | 條件描述 | 判斷依據 |
|------|---------|---------|
| M1 | 近期職稱包含機電/廠務/空調/水處理/監造/品管/採購/維運相關 | 搜尋「希望職稱」或工作經歷行 |
| M2 | 工作經歷中出現 EPC/營造/建廠/擴廠/半導體廠務/能源工程 | 搜尋公司名或職稱 |
| M3 | 有 3 年以上機電/廠務/工程相關實務年資 | 觀察工作經歷行數與年份差 |

### 2.3 排除條件（Exclusion）—— 符合任一項即排除

| 編號 | 條件描述 |
|------|---------|
| E1 | 經歷純粹為保全/門市/餐飲/銷售且無轉型跡象 |
| E2 | 希望職稱完全與工程/技術/機電/廠務無關 |
| E3 | 總工作年資不足 1 年且非應屆相關科系畢業 |
| E4 | 純土建人員且無建廠/機電/MEP經驗（土建人員必須具備建廠或跨MEP技術支撐） |
| E5 | 製程/製造/生產等非建廠工程人員（即使為機電相關，若為製造業製程人員亦不考慮） |
| E6 | 脫離高度工程專業（如純設備維修、保養、作業員等低階勞力工作） |

## 四、篩選經驗法則

> 以下為歷次疊代中累積的觀察與經驗，用於輔助 Agent 做軟性判斷。

### 4.1 學歷 vs 經驗
- **學歷非必要門檻**。非本科（如餐旅、資管、商業經營）但實務轉型成功者，只要有 5 年以上機電/廠務實戰經驗，仍可入選。
- 核心原則：**能力 > 學歷**。

### 4.2 年齡
- 不設嚴格年齡上限。歷史上 20~70 歲皆有被選入記錄。
- 但 **40~49 歲**為需求最大宗（約佔 36%），屬中堅技術人才。

### 4.3 年資甜蜜區
- **6~20 年**為最理想區間（約佔總選人之 68%）。
- 低年資（1-3 年）需具備「名校相關科系 + 半導體廠實習/正職」才有機會。
- 高年資（20 年+）需具備管理職或專家級經歷。

### 4.4 跨域整合價值
- 土木/建築背景但已轉型做機電、工地協調角色的人選具有獨特價值。
- 這類人適合「跨領域整合者」角色——能同時看懂土建圖與 MEP 圖。

### 4.5 營造工地管理角色
- Batch 2 確認：即使學歷僅國中/高職，只要擔任「機電主任」「工務主任」「工地主任」且年資充足（10年+），仍會被選入。
- 營造業的現場管理人員（監工、主任、襄理）是重要人才池，不可因學歷排除。

### 4.6 品管與採購也在範圍內
- 工程品管人員（品管工程師、品管人員）具備查驗、材料審核、計畫書編寫能力，屬需求範圍。
- 機電採購/發包角色（標單製作、議價、合約管理）也被選入，代表非純技術職也有需求。

### 4.7 能源轉型新趨勢
- Batch 2 出現太陽能（逆變器/儲能）、充電樁維運、高低壓電力維運等新興角色。
- 這類人選具備電力系統基礎且接觸新能源技術，符合產業趨勢。

### 4.8 軌道/捷運/公共工程
- Batch 2 新增桃園捷運綠線（大陸工程）、號誌系統（日立軌道）等軌道工程經驗。
- 公共工程品管（航廈、接收站）也在選人範圍內。
